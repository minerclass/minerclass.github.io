It connects Micah Miner's public research, writing, presentations, games, professional leadership work, and micahminer.com pathways in one searchable, filterable ecosystem.

## Coverage

- 46 featured live GitHub Pages sites
- 18 micahminer.com pages or current-series essays
- 64 searchable public entries
- 5 primary pathways: research, writing, presentations, games, and practice
